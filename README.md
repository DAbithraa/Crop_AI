CropGuard AI 🌱
Real-Time Crop Protection and Nutrient Advisory Web System
CropGuard AI is a smart agriculture web application developed using Java Spring Boot.
The system helps farmers identify crop diseases through image upload and provides detailed recommendations such as fertilizer dosage, maintenance tips, prevention methods, and crop health analysis.

📌 Features


🌿 Crop Disease Detection


📊 Crop Health Score


⚠ Disease Severity Level


💊 Solution Recommendation


🌱 Fertilizer Recommendation with Quantity


🛡 Prevention Methods


✅ Do’s and ❌ Don’ts


🌦 Weather-Based Advice


📍 Soil Recommendation


🗣 Tamil Language Advisory


📸 Image Preview Before Upload


🎤 Voice Output for Disease Result


🔐 Login Page Authentication


📂 Previous Result History


🎨 Green-Themed User Interface
